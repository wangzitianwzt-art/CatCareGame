# 🐱 猫咪养成插件 - 安装和使用指南

## 快速开始

### 方法 1：开发者模式安装（推荐用于测试）

1. **打开 Chrome 浏览器**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 点击右上角的「开发者模式」开关
   - 页面会显示三个新按钮

3. **加载已解压的扩展程序**
   - 点击「加载已解压的扩展程序」按钮
   - 选择项目中的 `dist` 文件夹
   - 点击「选择文件夹」

4. **完成！**
   - 插件现在应该出现在 Chrome 工具栏中
   - 点击插件图标即可打开猫咪养成游戏

### 方法 2：手动构建

如果您想从源代码构建：

```bash
# 1. 进入项目目录
cd /home/ubuntu/CatCarePlugin

# 2. 安装依赖
npm install

# 3. 构建插件
npm run build

# 4. 按照方法 1 的步骤 1-3 加载 dist 文件夹
```

## 使用方法

### 打开插件

1. **点击工具栏中的猫咪图标** 🐱
2. **弹窗会显示您的猫咪和游戏界面**

### 游戏操作

插件使用小图标进行交互，非常简洁：

| 图标 | 功能 | 说明 |
|------|------|------|
| 🍖 | 补充猫粮 | 点击打开输入框，输入数量后确认 |
| 🛁 | 给猫洗澡 | 每 24 小时限洗一次 |
| 🎾 | 逗猫咪 | 增加疲惫值，达到阈值自动睡眠 |
| 🔄 | 重置游戏 | 将游戏恢复到初始状态 |

### 游戏状态

弹窗显示三个实时状态指示条：

- **饥饿值**（红色）：0-100%
  - 补充猫粮后恢复
  - 猫粮为空时上升
  - 超过 70% 时猫咪不满

- **疲惫值**（橙色）：0-100%
  - 逗猫时上升
  - 睡眠时恢复
  - 超过 80% 时自动睡眠

- **清洁度**（绿色）：0-100%
  - 随时间减少
  - 洗澡时恢复到 100%

### 猫咪表情

猫咪会根据状态改变表情：

- 😸 **开心** - 正常状态
- 😾 **不满** - 饥饿值过高
- 😻 **疲惫** - 疲惫值过高
- 😴 **睡眠** - 自动睡眠中
- ✨ **清洁** - 刚洗过澡

## 数据保存

所有游戏数据都自动保存到 Chrome 本地存储中：

- 关闭浏览器后数据不会丢失
- 每次打开插件时自动恢复
- 点击 🔄 按钮可以重置所有数据

## 常见问题

### Q: 插件在哪里？
**A:** 点击 Chrome 工具栏右上角的拼图图标，找到"猫咪养成"，点击即可。

### Q: 如何固定插件到工具栏？
**A:** 在拼图菜单中，找到"猫咪养成"，点击旁边的图钉图标即可固定。

### Q: 游戏数据会被删除吗？
**A:** 
- 清除浏览器缓存时可能会删除
- 卸载插件时会删除
- 可以点击 🔄 按钮手动重置

### Q: 为什么猫咪自动睡眠了？
**A:** 疲惫值达到 80% 时，猫咪会自动进入 5 分钟的睡眠。睡眠期间无法进行任何操作。

### Q: 为什么不能洗澡？
**A:** 洗澡有 24 小时的冷却时间。每天只能洗一次澡。

### Q: 猫粮怎样消耗？
**A:** 
- 每 10 秒消耗 1 单位猫粮
- 猫粮为空时，饥饿值每 5 秒增加 1
- 及时补充猫粮可以防止猫咪饥饿

```

│   ├── assets/           # 图标资源
```

### 技术栈
- React 19
- TypeScript 5
- Vite 7
- Chrome Storage API

### 修改插件

如果您想修改插件：

1. **修改源代码** - 编辑 `src/` 目录中的文件
2. **重新构建** - 运行 `npm run build`
3. **刷新插件** - 在 `chrome://extensions/` 中点击刷新按钮
